## 📂 Repository Structure Example

```
algorithm-study
 ├─ BOJ
 │   ├─ 2563_색종이
 │   │   ├─ solution.py
 │   │   └─ README.md
 │   │
 │   ├─ 2178_미로탐색
 │   │   ├─ solution.py

 │
 └─ README.md
```

### 구조 설명

* **BOJ**
  백준 문제 풀이 모음

* **문제 폴더**

  ```
  문제번호_문제이름
  ```

* 각 문제 폴더에는 다음이 포함됨

| 파일          | 설명               |
| ----------- | ---------------- |
| solution.py | 문제 풀이 코드         |
| README.md   | 문제 접근 방법 및 학습 내용 |

## 🧠 Problem Solving Process

각 문제는 다음 기준으로 정리

1. **문제 요약**
2. **접근 방법**
3. **핵심 알고리즘**
4. **시행착오**
5. **배운 점**

---

## 🏷️ Algorithm Categories

문제 해결 과정에서 사용되는 주요 알고리즘 유형

* Brute Force
* Greedy
* DFS / BFS
* Dynamic Programming
* Graph
* Binary Search
* Two Pointer
* Sliding Window
* Sweep Line
* Coordinate Compression
* Data Structures (Stack / Queue / Heap / Set / Map)

## 🎯 Study Strategy

모든 문제를 기록하기보다 다음 문제들을 중심으로 정리

* 해결 과정이 어려웠던 문제
* 새로운 알고리즘 패턴이 등장한 문제
* 구현 과정에서 실수가 있었던 문제
* 복습 가치가 높은 문제
